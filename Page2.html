Création de site web
La réalité des développeurs dans l'industrie...

Un site internet, également appelé site web, est un ensemble de pages de textes, d'images et/ou de vidéos, reliées entre elles par des liens hypertextes également appelés liens internes. Ces éléments permettent de générer un affichage public pour les internautes désirant obtenir des réponses à leurs questions. Un site internet est donc une interface graphique accessible sur navigateur tel que Chrome, Firefox, Safari ou Brave. À ne pas confondre avec les moteurs de recherche tels que Google Bing où Yahoo. La réalisation d'un site web par un professionnel est avant tout un ensemble de programmations demandant la maîtrise de nombreux langages informatiques (PHP, JavaScript, Html5, CSS3, Curl, Soap, ...)
